## 🚀 快速开始

### 如何打开应用

双击应用的 HTML 文件即可在浏览器中打开使用。

### 系统要求

- 推荐使用现代浏览器（Chrome、Firefox、Edge、Safari 等）
- 建议使用 1920×1080 或更高分辨率的显示器
- 支持触摸屏设备，但推荐使用鼠标操作

### 功能说明

本编辑器提供了 ECharts 常用功能的可视化配置界面，涵盖了图表创建的核心功能，包括：
- 数据管理和编辑
- 基础图表配置（图例、提示框等）
- 坐标轴管理
- 布局调整
- 系列配置（支持折线图、柱状图、饼图、散点图）

如需使用 ECharts 的更多高级功能，可以通过代码编辑器直接修改配置。

## 🖥️ 界面介绍

应用界面主要分为两个部分：

### 左侧：编辑面板

编辑面板是您进行图表配置的主要区域，包含以下几个折叠面板：

1. **数据配置** - 管理图表的数据源
2. **基础配置** - 设置图例、提示框等基本功能
3. **坐标轴配置** - 管理 X 轴和 Y 轴
4. **布局配置** - 调整图表的位置和大小
5. **系列配置** - 添加和管理不同的图表类型

### 右侧：图表预览

图表预览区域实时显示您配置的图表效果，任何修改都会立即反映在这里。

## 📝 基本操作

### 1. 调整面板大小

您可以通过拖动左右面板中间的分隔线来调整编辑面板和预览面板的大小，找到最适合您的布局。

### 2. 折叠和展开面板

点击每个配置面板的标题可以折叠或展开该面板，方便您专注于当前需要修改的部分。

### 3. 查看配置代码

在编辑面板底部有一个代码编辑器，您可以查看当前图表的配置代码，也可以直接修改代码来调整图表。

## 🔧 功能详解

### 数据配置

#### 如何编辑数据

1. 点击"数据配置"面板右侧的设置图标（⚙️）
2. 在弹出的表格中，您可以：
   - **手动输入数据**：直接在表格单元格中输入数据
   - **复制粘贴数据**：从 Excel 或其他表格软件复制数据，粘贴到表格中
   - **调整表格大小**：输入行数和列数，点击"重置数据"按钮来调整表格尺寸

#### 数据格式说明

- 第一行通常是列标题（如：产品名称、年份等）
- 第一列通常是分类名称（如：产品名称）
- 其他单元格是对应的数据值

### 基础配置

#### 显示图例

- **开关**：控制是否显示图例
- **自动位置**：开启后，图例位置由系统自动计算（top 和 right 为 'auto'）
- **图例顶部位置**：关闭自动位置后，可以手动调整图例的顶部位置（0-100%）
- **图例右侧位置**：关闭自动位置后，可以手动调整图例的右侧位置（0-100%）

#### 显示标题

- **开关**：控制是否显示图表标题
- **标题文本**：输入图表的标题内容
- **自动位置**：开启后，标题位置由系统自动计算（top 和 right 为 'auto'）
- **标题顶部位置**：关闭自动位置后，可以手动调整标题的顶部位置（0-100%）
- **标题右侧位置**：关闭自动位置后，可以手动调整标题的右侧位置（0-100%）

#### 显示提示框

- 开启后，鼠标悬停在图表上时会显示详细数据信息
- 提示框会自动根据图表类型调整显示方式

### 坐标轴配置

#### X 轴管理

- **添加 X 轴**：点击"新增 X 轴"按钮添加新的 X 轴
- **删除 X 轴**：点击 X 轴项左侧的删除按钮（-）
- **配置 X 轴**：点击 X 轴项右侧的设置图标（⚙️）进行详细配置

#### Y 轴管理

操作方式与 X 轴相同，用于管理垂直方向的坐标轴。

#### 坐标轴配置选项

- **图表类型**：选择坐标轴的类型（分类轴、数值轴等）
- **轴名称**：为坐标轴设置显示名称
- **轴位置**：设置坐标轴在图表中的位置

### 布局配置

#### Grid 位置调整

- **顶部位置**：调整图表区域距离顶部的距离
- **左侧位置**：调整图表区域距离左侧的距离
- **宽度**：调整图表区域的宽度
- **高度**：调整图表区域的高度

这些调整使用滑块控制，拖动滑块即可实时看到效果。

### 系列配置

#### 添加图表系列

1. 点击"新增系列"按钮
2. 新的系列会出现在列表中
3. 点击系列右侧的设置图标（⚙️）进行配置

#### 图表类型

支持以下图表类型：

- **折线图**：适合展示趋势变化
- **柱状图**：适合比较不同类别的数据
- **饼图**：适合展示占比关系
- **散点图**：适合展示两个变量之间的关系

#### 系列配置选项

- **系列名称**：为该系列设置一个名称（饼图除外）
- **绑定 X 轴**：选择该系列使用哪个 X 轴
- **绑定 Y 轴**：选择该系列使用哪个 Y 轴
- **X 映射**：选择数据源中的哪一行作为 X 轴数据（显示行名称，如 "Milk Tea"、"Matcha Latte" 等）
- **Y 映射**：选择数据源中的哪一行作为 Y 轴数据（显示行名称，如 "Milk Tea"、"Matcha Latte" 等）

#### 饼图特殊配置

对于饼图类型，还可以配置：

- **数据列**：选择数据源中的哪一列作为饼图的数据（显示列名，如 "2012"、"2013" 等）
- **饼图半径**：调整饼图的大小
- **中心位置 X**：调整饼图中心的水平位置
- **中心位置 Y**：调整饼图中心的垂直位置

## 💡 使用技巧

### 1. 快速创建图表

1. 先在"数据配置"中输入或粘贴您的数据
2. 在"系列配置"中添加一个图表系列
3. 在系列配置中选择合适的图表类型和数据映射
4. 图表会自动显示在右侧预览区域

### 2. 组合多种图表类型

您可以在同一个图表中组合多种图表类型，例如：

- 使用折线图展示趋势
- 使用柱状图对比数据
- 使用饼图展示占比

只需添加多个系列，并为每个系列配置不同的图表类型即可。

### 3. 调整图表布局

如果图表显示不够美观，可以：

1. 在"布局配置"中调整 Grid 的位置和大小
2. 在"基础配置"中调整图例位置
3. 在"坐标轴配置"中调整坐标轴的显示方式

### 4. 使用代码编辑器

如果您熟悉 ECharts 配置，可以直接在代码编辑器中修改配置：

1. 在编辑面板底部找到代码编辑器
2. 修改相应的配置代码
3. 点击"运行代码"按钮应用更改

## ❓ 常见问题

### Q1: 如何保存我的图表？

A: 当前版本主要提供在线编辑功能。如需保存，建议：
- 复制配置代码，保存到文本文件中
- 右键点击图表区域，选择"图片另存为"保存为图片格式

### Q2: 图表显示不正常怎么办？

A: 请检查以下几点：
- 数据格式是否正确（第一行是否为标题，第一列是否为分类）
- 系列配置中是否正确绑定了数据映射
- 坐标轴配置是否与图表类型匹配

### Q3: 如何调整图表的颜色？

A: 当前版本使用默认配色。如需自定义颜色，可以在代码编辑器中修改 series 配置的 color 属性。

### Q4: 支持哪些数据格式？

A: 支持标准的表格格式数据：
- 第一行为列标题
- 第一列为分类名称
- 其他单元格为数值数据

### Q5: 可以创建多少个系列？

A: 没有限制，您可以根据需要添加任意数量的图表系列。

### Q6: 图表显示太小或太大怎么办？

A: 在"布局配置"中调整 Grid 的宽度和高度，或者调整浏览器的缩放比例。

### Q7: 如何删除不需要的系列？

A: 在"系列配置"中，点击系列左侧的删除按钮（-）即可删除该系列。

### Q8: 编辑器支持哪些 ECharts 功能？

A: 编辑器提供了 ECharts 常用功能的可视化配置，包括数据管理、基础配置、坐标轴管理、布局调整和系列配置等核心功能。如需使用更多高级功能，可以通过代码编辑器直接修改配置。
