## 创建 ECharts 编辑器静态页面

基于对原页面的分析，我将创建一个静态 HTML 页面来复现其布局结构：

### 实现内容：

1. **HTML 结构**
   - 顶部固定导航栏（50px 高度）
   - 左右分栏主容器
   - 左侧编辑器面板（40% 宽度）
   - 右侧图表预览面板（60% 宽度）

2. **左侧编辑器面板组件**
   - 标签页导航（代码编辑、完整代码、配置项）
   - 语言切换按钮（JS/TS）
   - 工具按钮组（CodePen、CodeSandbox、格式化、运行）
   - 代码编辑器区域（使用 textarea 模拟）

3. **右侧图表预览面板**
   - 图表展示区域
   - 示例图表占位符

4. **CSS 样式**
   - 使用 Flexbox 实现响应式布局
   - 复制原页面的配色方案
   - 添加适当的间距和边框
   - 使用 Bootstrap 风格的按钮样式

5. **技术栈**
   - 纯 HTML + CSS
   - 不使用 JavaScript 框架
   - 可选：引入 Bootstrap CSS 库以快速实现样式

### 文件结构：
- `echarts-editor.html` - 主页面文件
- `echarts-editor.css` - 样式文件（可选，内联到 HTML 中）

这个静态页面将用于验证我对原页面布局的理解是否正确。
