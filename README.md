# ECharts 图表编辑器

## 项目介绍

ECharts 图表编辑器是一个基于 React 和 ECharts 的可视化图表配置工具，提供了 ECharts 常用功能的可视化配置界面，帮助用户更便捷地创建和定制图表。

### 项目背景

本项目使用 AI IDE Vibe Coding 进行开发，前期通过 trae solo 模式进行功能规划，项目功能基本确定后，边体验边使用 AI Prompt 功能进行修改和优化。

### 前期规划

项目前期规划文件存储在 `.trae/` 目录中，详细记录了项目的设计思路和功能规划。

### 功能说明

本编辑器提供了 ECharts 常用功能的可视化配置界面，涵盖了图表创建的核心功能，包括：
- 数据管理和编辑
- 基础图表配置（图例、提示框等）
- 坐标轴管理
- 布局调整
- 系列配置（支持折线图、柱状图、饼图、散点图）

如需使用 ECharts 的更多高级功能，可以通过代码编辑器直接修改配置。

## 运行和构建指南

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

开发服务器会在 http://localhost:3000 启动，并自动打开浏览器。

### 构建生产版本

```bash
npm run build
```

构建产物会输出到 `dist` 目录。

### 构建离线版本

```bash
npm run build:offline
```

离线版本会将所有资源内联到单个 HTML 文件中，适合在没有网络环境的情况下使用。

### 预览构建结果

```bash
npm run preview
```

## 项目特点

1. **可视化配置**：提供直观的可视化界面，无需编写代码即可配置图表
2. **多图表类型支持**：支持折线图、柱状图、饼图、散点图等多种图表类型
3. **数据管理**：内置表格编辑器，方便用户输入和修改数据
4. **代码编辑**：支持直接修改配置代码，满足高级用户需求
5. **离线使用**：支持构建单文件离线版本，可在无网络环境下使用
6. **响应式设计**：适配不同屏幕尺寸

## 技术架构

- **前端框架**：React 18
- **构建工具**：Vite 5
- **类型系统**：TypeScript
- **图表库**：ECharts 5
- **UI组件**：Semi-UI
- **路由管理**：React Router (HashRouter)
- **数据管理**：React Spreadsheet
- **代码编辑器**：CodeMirror 6
- **单文件构建**：Vite Plugin SingleFile

## 浏览器支持

- 推荐使用现代浏览器（Chrome、Firefox、Edge、Safari 等）
- 建议使用 1920×1080 或更高分辨率的显示器
- 支持触摸屏设备，但推荐使用鼠标操作
